## How to build

- `npm run build` — Copy required files in `dist/luatools`.
- `npm run build:zip` — Same as build but also create `dist/luatools.zip`.

## How to clone

```bash
# Repo clone:
git clone git@github.com-luatools:muttape/luatools.git

# Add upstream
git remote add upstream git@github.com:madoiscool/ltsteamplugin.git

# Disable push to upstream:
git remote set-url --push upstream DISABLE
```
